# Фото-оверлей на весь интерфейс

Меню **💥 ПИЗДЕЦ → 🐐 Фото на весь экран (вращение+пульс)** показывает
картинку поверх всего интерфейса: она вращается и плавно пульсирует
прозрачностью (мягко, без резкого стробоскопа).

Чтобы использовать своё фото (например, того самого Kozel), положи файл сюда
под одним из имён (берётся первый существующий):

1. `images/kozel_overlay.jpg`
2. `images/kozel_overlay.png`
3. `images/eblan_overlay.png`
4. `images/eblan_overlay.jpg`

Если ни одного нет — используется фолбэк `images/eblanai.png`, чтобы
функция работала сразу.

Оверлей клик-сквозной: браузером под ним можно пользоваться.
